# AprendiendoMuchoGit

## Actividad de Git y GitHub

Durante esta actividad se ha trabajado con **Git**, *GitHub* y `Visual Studio Code` para gestionar un proyecto de programación, controlando versiones, creando ramas y sincronizando cambios con un repositorio remoto.

```bash
git init
git status
git add .
git commit -m "Primer commit del proyecto"
git push origin main
```

1. Crear un repositorio en GitHub
2. Clonar el repositorio en Visual Studio Code
3. Realizar commits y subir cambios

- Uso de Git desde la terminal
- Trabajo con ramas en GitHub
- Sincronización con repositorio remoto
